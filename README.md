# TheDreamer Tap

## How do I install these formulae?

`brew install thedreamer/tap/<formula>`

Or `brew tap thedreamer/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "thedreamer/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
